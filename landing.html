Telescope Scheduling Intelligence

Modern web application for analyzing and visualizing astronomical scheduling outputs with a high-performance Rust backend and Vue.js frontend

🦀 Rust + Axum ⚡ Polars 🎨 Vue 3 📘 TypeScript 🐳 Docker

✨ Key Features

🗺️

Sky Map Visualization

Interactive RA/Dec scatter plots with color-coded priorities and scheduling status

📊

Statistical Analysis

Comprehensive distributions of priority, visibility hours, and observation durations

📅

Timeline Views

Month-by-month scheduling timeline with dark/daytime period overlays

🔍

Insights & Trends

Scheduling rate analysis, correlations, and integrity checks with temporal evolution

⚡

High Performance

Rust backend with Polars for blazing-fast data processing and analysis

🔄

Data Preprocessing

JSON/CSV loaders with automated preprocessing pipeline for consistent data handling

🚀 Getting Started

⚠️ Setup Required
Your system needs some dependencies to run TSI. The application requires either Docker or manual installation of Rust and Node.js.

📦 Option 1: Docker (Recommended)

# Install Docker Compose
sudo apt install docker-compose

# Start the application
docker-compose up --build

After starting, access the frontend at http://localhost:5173

🔧 Option 2: Manual Setup

Install the required dependencies:

# 1. Update Rust (current: 1.75.0, needed: 1.76+)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup update

# 2. Install Node.js
sudo apt install nodejs npm

# 3. Start Backend (Terminal 1)
cd backend && cargo run --release

# 4. Start Frontend (Terminal 2)
cd frontend && npm install && npm run dev

🌐 Access Points

  • Frontend: http://localhost:5173
  • Backend API: http://localhost:8080 or :8081
  • Health Check: http://localhost:8080/health
  • API Docs: http://localhost:8080/swagger-ui
